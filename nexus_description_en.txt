[left]
[list]
[*] [b]Easy Game Launch[/b]: Launch Stardew Valley in either Vanilla mode or through SMAPI for modded play.
[*] [b]Mod Manager[/b]: Enable or disable mods effortlessly through a beautiful app interface — no manual file moving required.
[*] [b]Mod Profiles[/b]: Group mods into multiple profiles and switch between them instantly with a single click.
[*] [b]Thai Translation Hub[/b]: A dedicated hub listing all Thai translation mods — browse, check status, download, and track updates in one place.
[*] [b]Save Manager[/b]:
    [list]
    [*] View details of all save files (money, in-game time, season, farm layout)
    [*] Duplicate or delete save files
    [*] Edit money and basic character stats
    [/list]
[*] [b]Developer Logs[/b]: Monitor SMAPI output in real time directly within the app.
[*] [b]Bilingual Support[/b]: Switch the app language instantly between English and Thai (ภาษาไทย).
[*] [b]Native macOS UI[/b]: A clean, intuitive interface designed to feel right at home on macOS.
[/list]
[/left]

[list=1]
[*] [b]Download[/b]: Grab the latest release from the Files tab on Nexus Mods.
[*] [b]Install[/b]: Unzip the file and drag [b]StarHubTH.app[/b] into your Applications folder, then double-click to launch.
[*] [b]Set Game Folder[/b]: On first launch, the app will attempt to auto-detect your Steam game folder. If not found, you can manually select the game directory (e.g. /Applications/Stardew Valley.app/Contents/MacOS).
[*] [b]You're ready![/b]: Manage your mods or saves, then hit [b]"Launch Game"[/b] from the left sidebar.
[/list]
